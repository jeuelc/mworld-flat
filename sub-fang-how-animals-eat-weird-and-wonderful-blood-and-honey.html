Can you imagine only ever eating through a straw? Maybe you've had a sore tooth and not been allowed to chew food, so would know what it's like. But there are many animals that only feed on liquids.

Vampire bats only ever drink blood, which is the best food there is because it has all you need to live. The bats do this by using such sharp teeth to make a cut in a cow or dog that the animal doesn't even notice, and it starts to bleed. The bat laps up all this blood, and sometimes drinks so much it's too heavy to fly! It has to walk away and digest all that blood before it can fly again. Mosquitoes suck blood using very long needle-like mouthparts that pierce the skin of its victim.

Butterflies drink the liquid nectar in flowers, the same stuff used by bees to make honey. A butterfly drinks nectar using a really long straw that it rolls up when it's not using it!
